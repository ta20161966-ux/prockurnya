# GitHub User Finder

## Автор
Иванов Иван

## Описание
Приложение для поиска пользователей GitHub с возможностью добавления в избранное и сохранением данных в JSON.

## Использование API
Для поиска используется GitHub API: GET https://api.github.com/users/{username}

## Примеры использования
1. Введите логин пользователя в поле поиска и нажмите «Поиск».
2. Кликните «В избранное», чтобы сохранить пользователя.
3. Избранные пользователи сохраняются в favorites.json.
